# 📊 Data Mining Project — Book Client Prediction

A data mining project that analyzes client booking behavior and builds a classification model to predict whether a client will make a booking.

---

## 📁 Repository Contents

| File | Description |
|------|-------------|
| `Data_mining_Project_Book_clients.ipynb` | Main Jupyter Notebook containing the full analysis and model |
| `Book client prediction.docx` | Project report / documentation |

---

## 🎯 Objective

Predict whether a client will book a service based on their behavioral and demographic attributes, using supervised machine learning (Decision Tree Classification).

---

## 🔍 Project Workflow

1. **Data Loading** — Import and inspect the raw dataset
2. **Data Cleaning** — Handle missing values, duplicates, and inconsistencies
3. **Label Encoding** — Convert categorical features to numerical format
4. **Exploratory Data Analysis (EDA)** — Visualize distributions, correlations, and booking patterns
5. **Model Building** — Train a Decision Tree Classifier to predict booking outcomes
6. **Evaluation** — Assess model performance using accuracy and classification metrics
7. **Decision Rule Visualization** — Plot the decision tree to interpret learned rules
